# 📊 Dynamic Retail Dashboard|(Excel)

## 📌 Project Overview

This project is a dynamic and interactive **** created in **Dynamic Retail Dashboard** created in **Microsoft Excel** using the Walmart retail dataset. The dashboard helps analyze sales performance, profitability, customer behavior, product categories, and regional trends to generate meaningful business insights.

The main objective of this project is to transform raw retail data into an easy-to-understand dashboard that supports data-driven decision-making.

---

# 🎯 Business Problem Statement

With growing competition in the retail industry, Walmart wants to understand:

- Which categories generate the highest sales and profit
- Which customer segments contribute the most revenue
- Which regions and countries perform better
- Which categories are underperforming
- How sales are growing year by year

The dashboard helps identify opportunities for improving business performance and operational efficiency.

## 3️⃣ Returns Table

# ▶️ How to Use

1. Download the Excel dashboard file from the dataset link above.

5. Analyze:
- Sales Performance
- Profit Trends
- Category Performance
- Regional Sales
- Customer Segments

6. Navigate through different dashboard sections to explore insights.
